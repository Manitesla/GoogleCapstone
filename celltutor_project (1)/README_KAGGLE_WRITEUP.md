# Agent CellTutor — Capstone Project Writeup

*Agent CellTutor automates per-cell teaching for interactive notebooks: explanations, visuals, and quizzes.*

## Project Overview

Agent CellTutor is a multi-agent instructional system designed to convert notebook code cells into interactive, multimodal mini-tutors. Each cell agent explains code, generates visuals (diagrams, animations), answers questions, and quizzes the learner — storing results in a registry.
